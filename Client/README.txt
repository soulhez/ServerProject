		   
本项目主要实现：客户端向服务器发送过来的XML、JSON、bmp格式的图片、txt格式的文件、URL视频链接等，客户端对这些数据通过数据包的形式进行发送，服务器和客户端通过交互来实现加密或不加密传输。服务器同样进行解密或不解密接收。

主要操作步骤：
1、将本文件复制到Linux系统的目录下

2、编译 gcc Client.c -o main -lcrypto

3、执行 ./main 6000 10.10.121.201

4、与服务器连接成功后，会出现一个菜单，
		0：请求发送数据包 1：断开连接 2：发送本地xml文件
		3：发送本地json文件 4：发送本地bmp格式图片
		5：发送本地txt格式文件 6：发送URL视频链接
	
5、客户端会根据自己输入的命令 来发送对应的文件数据包
