		   
本项目主要实现：服务器接收客户端发送过来的XML数据、JSON数据、bmp格式的图片、txt格式的文件、URL视频链接等，客户端对这些数据通过数据包的形式进行发送，服务器和客户端通过交互来实现加密或不加密传输。服务器同样进行解密或不解密接收。

主要操作步骤：
1、在windows下按住：CTRL + r 输入 cmd 进入dos界面， 通过cd命令进行本项目 jni所在路径下
	如：E: => 进入E盘  cd E:\code\serverProject => 进到jni所在路径

2、输入 ndk-build 对整个项目进行编译

3、用tftp32工具 将 该项目路径 libs/armeabi/ 下的 main文件上传到机顶盒中
	用SecureCRT链接机顶盒在输入命令 如： tftp -gr main 10.10.121.62(服务器IP地址)
	将main文件下载到盒子中
	
4、给main文件加权限  chmod 777 main

5、执行 ./main

6、与客户端连接成功后，当用户输入请求数据包时，服务器会打印请求信息
	如：输入0 拒绝； 输入 1 不加密； 输入 2 加密
	
7、服务器同意客户端连接后，服务器会根据所接收到的命令类型来对应进行文件存储或数据解析或视频下载

8、不同的客户端 会在服务器本地路径下生成不同的文件夹，来存储接收到的文件

本项目包括7个.cpp源文件
1、main.cpp 程序入口
2、ServerRecv.cpp 负责接收客户端的数据包，根据命令类型调用不同类的成员函数
3、ServerTools.cpp 主要负责为其他类服务，提供一些方法，使得操作更简单
4、ServerSend.cpp 主要是发送反馈数据包给客户端
5、ServerFile.cpp 负责解析和保存各类文件
6、ServerSocket.cpp 主要是初始化服务器的一些信息，为客户端创建线程完成对应操作
7、ServerAes.cpp  主要是负责解密客户端发过来的加密数据包
